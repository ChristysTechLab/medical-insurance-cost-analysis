# Medical Insurance Cost Analysis

## Project Overview
This project explores a medical insurance dataset to analyze factors that influence individual insurance costs. Using exploratory data analysis (EDA), this project examines relationships between demographic and lifestyle factors and insurance charges.

## Objectives
- Analyze patterns in medical insurance costs
- Explore how factors such as age, BMI, smoking status, and region impact charges
- Identify trends and relationships within the dataset
- Create visualizations to communicate findings

## Dataset
The dataset contains medical insurance information including:
- Age
- Sex
- BMI
- Number of children
- Smoking status
- Region
- Insurance charges

## Dataset Source
Medical Cost Personal Dataset from Kaggle.
